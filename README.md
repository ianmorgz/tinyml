# TinyML - Embedded Neural Network Inference Toolchain
## What is TinyML?
This project is an end-to-end pipeline for deploying neural networks on microcontrollers. It trains and quantized fully connected neural networks on a hist computer in C++, then generates static C code for inference within your embedded ARM projects. 
* Easy to use: TinyML is designed with a straight forward API and class based library design which can easily allow you to take full advantage of existing features or add your own.
* Customizable training and quantization: Don't know what you want for a network? 
* Allocation free: TinyML's generated C code is free of dynamic memory which allows a small footprint to run alongside your project.
* Drag and drop design: Code generated with TinyML can simply be copied into your existing embedded projects, with a just two callable functions to set up and run, allowing for it to be used at your own digression.
